# REPLIT AGENT INSTRUCTIONS: Alliance Trust Realty Website Rebuild

## PROJECT OVERVIEW

Rebuild Alliance Trust Realty website as a professional, full-service real estate platform specializing in investment properties. The site must serve both traditional homebuyers/sellers AND real estate investors with distinct user pathways and features.

-----

## TECHNICAL FOUNDATION

### Stack Requirements

- Frontend: React with TypeScript
- Styling: Tailwind CSS for modern, responsive design
- Forms: React Hook Form with validation
- Icons: Lucide React
- Animations: Framer Motion for smooth transitions
- State Management: React Context or Zustand for global state
- NO localStorage/sessionStorage (use React state only)

### Performance Requirements

- First Contentful Paint < 1.5s
- Lazy load images and components
- Optimize images (WebP format, responsive sizes)
- Implement code splitting
- Mobile-first responsive design
- Accessibility: WCAG 2.1 AA compliance

## DESIGN SPECIFICATIONS

### Color Palette

```css
Primary: #1e40af (Trust blue)
Secondary: #059669 (Growth green)
Accent: #dc2626 (Urgency red for CTAs)
Neutral Dark: #1f2937
Neutral Light: #f3f4f6
White: #ffffff
```

### Typography

- Headings: Inter or Poppins (bold, modern)
- Body: Inter or System UI (readable)
- Numbers/Stats: Tabular figures for alignment

### Component Style Guidelines

- Cards: Subtle shadows, rounded corners (8px)
- Buttons: Primary (filled), Secondary (outline), Ghost
- Forms: Clear labels, inline validation, helpful error messages
- Images: Aspect ratio 16:9 for properties, 1:1 for team photos
- Spacing: Consistent 8px grid system
- Hover states: Smooth transitions (200-300ms)

### Mobile Responsiveness Breakpoints

- Mobile: < 640px (stack everything)
- Tablet: 640px - 1024px (2 columns)
- Desktop: > 1024px (3-4 columns)
- Large: > 1280px (max content width 1280px, center)

## SAMPLE CALCULATOR IMPLEMENTATION

```jsx
// ROI Calculator Component
const ROICalculator = () => {
  const [inputs, setInputs] = useState({
    purchasePrice: 0,
    downPayment: 20,
    interestRate: 7.0,
    loanTerm: 30,
    monthlyRent: 0,
    propertyTax: 0,
    insurance: 0,
    hoa: 0,
    maintenance: 0,
    vacancy: 5
  });

  const calculate = () => {
    // Loan calculations
    const loanAmount = inputs.purchasePrice * (1 - inputs.downPayment / 100);
    const monthlyRate = inputs.interestRate / 100 / 12;
    const numPayments = inputs.loanTerm * 12;
    const monthlyPI = loanAmount * (monthlyRate * Math.pow(1 + monthlyRate, numPayments)) / 
                      (Math.pow(1 + monthlyRate, numPayments) - 1);
    
    // Monthly expenses
    const monthlyTax = inputs.propertyTax / 12;
    const monthlyInsurance = inputs.insurance / 12;
    const totalMonthlyExpenses = monthlyPI + monthlyTax + monthlyInsurance + 
                                 inputs.hoa + inputs.maintenance;
    
    // Cash flow
    const effectiveRent = inputs.monthlyRent * (1 - inputs.vacancy / 100);
    const monthlyCashFlow = effectiveRent - totalMonthlyExpenses;
    const annualCashFlow = monthlyCashFlow * 12;
    
    // Returns
    const totalInvestment = inputs.purchasePrice * (inputs.downPayment / 100);
    const cashOnCashReturn = (annualCashFlow / totalInvestment) * 100;
    const capRate = ((inputs.monthlyRent * 12) - 
                    (inputs.propertyTax + inputs.insurance + (inputs.hoa * 12) + 
                     (inputs.maintenance * 12))) / inputs.purchasePrice * 100;

    return {
      monthlyPI,
      monthlyCashFlow,
      annualCashFlow,
      cashOnCashReturn,
      capRate,
      totalInvestment
    };
  };

  const results = calculate();

  return (
    <div className="calculator-container">
      <h2>Investment ROI Calculator</h2>
      
      <div className="inputs-grid">
        {/* Input fields */}
      </div>
      
      <div className="results-grid">
        <ResultCard 
          label="Monthly Cash Flow" 
          value={results.monthlyCashFlow} 
          format="currency"
          positive={results.monthlyCashFlow > 0}
        />
        <ResultCard 
          label="Cash-on-Cash Return" 
          value={results.cashOnCashReturn} 
          format="percentage"
        />
        <ResultCard 
          label="Cap Rate" 
          value={results.capRate} 
          format="percentage"
        />
        <ResultCard 
          label="Annual Cash Flow" 
          value={results.annualCashFlow} 
          format="currency"
        />
      </div>
      
      <Button>Export Results</Button>
    </div>
  );
};
```
